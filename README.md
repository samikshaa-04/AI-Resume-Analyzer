# 🤖 AI Resume Analyzer & Job Recommendation System

## 📌 Project Overview

The AI Resume Analyzer & Job Recommendation System is a web application that analyzes a user's resume, extracts technical skills, compares them with job requirements, identifies missing skills, and recommends suitable job roles. It also provides personalized suggestions to improve the resume and increase job readiness.

---

## ✨ Features

- 📄 Upload resume in PDF format
- 🔍 Extract text from the resume
- 🧠 Detect technical skills using AI/NLP
- 📊 Calculate resume score
- 📈 Perform skill gap analysis
- 💼 Recommend suitable job roles
- 💡 Provide resume improvement suggestions
- ⚡ Simple and user-friendly interface

## 🚀 How to Run the Project

### Step 1: Clone the Repository

```bash
git clone https://github.com/samikshaa-04/AI-Resume-Analyzer.git
```

### Step 2: Move into the Project Folder

```bash
cd AI-Resume-Analyzer
```

### Step 3: Install the Required Libraries

```bash
pip install -r requirements.txt
```

### Step 4: Run the Flask Application

```bash
python app.py
```

### Step 5: Open in Browser

```
http://127.0.0.1:5000
```
